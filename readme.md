# Docker Compose Setup

## Usage

### Default Setup
Run:
```sh
docker-compose up -d
```

### Development Setup
Run:
```sh
docker-compose -f docker-compose-dev.yaml up -d
```

### Stop Containers
```sh
docker-compose down
```
For dev setup:
```sh
docker-compose -f docker-compose-dev.yaml down
```

## Notes
- Ensure Docker and Docker Compose are installed.
- Use `sudo` if permission issues arise.
